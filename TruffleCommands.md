# CREATING CONTRACT
1. truffle init -> empty project
2. truffle create contract [name] -> creating contract
3. Add migration in js
4. truffle migrate

# CREATING FRONT IN SVELTE
1. npm create svelte@latest app-name
2. cd myapp
3. npm install
4. npm run dev || npm run dev -- --open
